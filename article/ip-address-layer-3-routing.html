IP Address - Layer 3 Routing

Commands

  • ifconfig

inet is ipv4

  • notation is in decimal format (i.e. 172.16.12.129)

inet6 is ipv6

  • notation is in hexdecimal format (fe80::20c:29ff:fe55:a2a4)

Both "communicate on Layer 3"

255 64 32 16 8 4 2 1 = 255
1 1 1 1 1 1 1 1 = 255
0 0 0 0 0 1 1 1 = 7
Computers are just ones and zeros, they are binary
